Main part of the screen is a map where the user can indicate start and end points for their pipes. After that they need to setup the pipe zone profile.



To decrease chances of the walls caving in, the top part of the trench is larger than the bottom. Here the user can also indicate pipe size and the bedding dimensions. The backfill depths will be calculated from the DEM and the start and end locations that user indicated.



After the pipeline has been submitted, the viewer can see the side profile of the pipe. This will show the existing surface elevations with the desired pipe elevations. It will also indicate how much soil needs to be excavated.



 

The summary table will shows the volume of excavation as well as how much backfill and bedding import is needed based on the pipe zone details.
