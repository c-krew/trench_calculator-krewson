Data Services!


Data Services: The main data service that we have used so far is the ArcGIS server. This server allows us to store shapefiles and then retrieve them inside of our application.

Data Creation: The data that we are using was found from gis.utah.gov. We clipped the municipalities data so that it only showed Utah valley.

Data Hosting: The url for the Geoserver that we are using is http://geoserver2.byu.edu/arcgis/rest.

Data Types: The two types of data that we are using is a DEM and a vector shapefile of the municipalities in Utah Valley
